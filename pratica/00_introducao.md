# Introdução à prática

Podemos utilizar tanto o arquivo `.ipynb` (python notebook) quanto o `.py` usando as separações com `# %%` para criar as células como se fosse notebook.
Exemplo:
```python
# %% Bibliotecas
....

# Seção 2
.... 
```

OBS: Com o `.py`, o versionamento do código é mais fácil, uma vez que com o `.ipynb`, para cada run que damos, o código atualiza, além de que o que é colocado em produção é o `.py`.
